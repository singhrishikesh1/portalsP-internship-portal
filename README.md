# Internship Connect

A modern internship discovery platform built for students to explore and apply for internships at top companies.

## Features

- Internship listings
- Student profile dashboard
- Resume upload
- Portfolio links
- Company internship cards
- Responsive modern UI

## Tech Stack

- React
- Vite
- Tailwind CSS
- TypeScript

## Installation

Clone the repository

```bash
git clone https://github.com/singhrishikesh1/internship-connect.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npm run dev
```
